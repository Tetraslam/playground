# RESOURCES.md — agent-contributed links

This is the **shared/agent** link pile. tetraslam's personal curated collection
lives at <https://tetraslam.world/links> and stays separate (read-only upstream
— don't add to it). Links *agents* find go here instead.

Browse both sources merged with `tools/links.sh` (filter by tag). Add a link
here with:

```bash
tools/links.sh --add ml,rl "https://arxiv.org/abs/..." "why it matters"
```

Format per entry: `### title` / `**Tags**: a, b` / `**URL**: ...` — same shape
as the upstream feed so the two merge cleanly.

Left-click empty space to place the start. Right-click for the goal. Drag in empty space to draw walls. Click an existing wall to erase it. The tree grows until it finds the goal — then a path lights up.

The tree samples the world randomly, then extends a branch from its nearest node toward the sample — clipping at walls. RRT* rewires the tree around each new node to find shorter paths. A* is the deterministic grid-search classic: glowing frontier, then a clean path. Try the narrow corridor scenario: A* crawls through, RRT* takes longer to notice the gap.

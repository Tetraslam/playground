# svgart

SVG art generated programmatically. The canvas is the medium; Python is the brush.

## drift_panorama

A cross-section panorama of **the Drift** — the alien world from the bestiary100
toy. All 8 biomes stacked vertically (Rime through the Vent), with creatures
appearing in their native habitats, atmospheric effects (aurora, bioluminescence,
thermal glow, heat shimmer), light shafts, depth fog, ambient particles, and a
decorative border.

1600 × 3200px. ~470KB of SVG. Generated by a Python script that uses math and
loops for fractal detail — no hand-typed coordinates.

### how to run

```bash
uv run --no-project toys/svgart/drift_panorama.py
# output: toys/svgart/examples/drift_panorama.svg
```

Convert to PNG for viewing:
```bash
rsvg-convert toys/svgart/examples/drift_panorama.svg -o drift.png -w 800
```

### how it works

The generator (`drift_panorama.py`) builds SVG from pure Python — no external
libraries, no SVG templates. Helper functions (`path`, `circle`, `ellipse`,
`rect`, `line`, `polygon`, `group`) wrap raw SVG element construction. Gradients
and filters (glow, blur, crystal refraction) are defined once in `<defs>` and
referenced by URL.

Each biome is a separate render function that:
1. paints a gradient background
2. adds terrain/structures (ice formations, tree trunks, fungal columns, dunes, fissures)
3. places creatures with anatomy matching the bestiary descriptions
4. layers atmospheric effects (particles, mist, glow, shimmer)

A final pass adds light shafts crossing biome boundaries, depth fog at
transitions, ambient particles, and a decorative border.
